kerangka website ada tiga, yaitu HTML sebagai kerangka (tempat merancang halamannya), CSS sebagai kulit (untuk mempercantik tampilan), dan Javascript sebagai otaknya (untuk menyimpan logika logikanya). cara browser bekerja : ketika mengetik www.dicoding.com, nanti browser akan merequest alamat ip dari www.dicoding.com itu, lalu setelah alamat ip nya dapat, browser akan melakukan request ke server untuk menampilkan halaman yang ada di alamat ip tersebut. lalu web server akan diproses lalu memberikan respons ke browser. HTML adalah sebuah markup code untuk membuat sebuah struktur halaman web dan juga kontennya yang dikembangkan oleh W3C. pertama kali dikenalkan oleh Tim Berners-lee pada tahun 1993. website pertama berisi heading, paragraf, dan juga link. versi HTML sekarang tidak memiliki versi angkanya karena selalu diupdate. anatomi halaman HTML secara umum adalah tag pembuka, atribut, konten, terakhir tag penutup. semantic elemen adalah memberikan struktur di dokumen html yang jelas. Semantic element lebih mudah dibaca. Penggunaan semantic element mempunyai beberapa keunggulan, yaitu meningkatkan experience developer (biar developer mudah ketika proses development), meningkatkan SEO (ketika menggunakan struktur yang jelas, search engine lebih mudah mengetahui strukturnya), meningkatkan aksesibilitas. 

CSS adalah singkatan dari cascading style sheet (runtutan baris baris styling). dikembangkan oleh W3C juga, tetapi memiliki grup yang berbeda dari html, yaitu css working group. Cascading stylesheet ini bisa digunakan untuk memberikan gaya atau style dari dokumen atau halaman html, bisa mengubah gaya, font yang digunakan, ukuran font yang digunakan, ukuran layar, warna teks, warna background, layout, bahkan animasi. anatomi dari css adalah selector{property : value}. selector adalah apa yang akan di assign ke element html tersebut. selector dapat berupa tag, class, atau id dari element html. property adalah apa yang akan digunakan, misalnya color untuk mengubah warna teks. sedangkan valuenya adalah nilai yang akan diisi, misalnya green akan mengubah warnanya menjadi hijau. contohnya dari ruleset CSS adalah seperti berikut: p{color:red;), artinya seluruh dokumen html yang memiliki tag <p> di html, seluruh teks nya akan berwarna merah. 

Responsive web design adalah sebuah fitur yang baru dikembangkan, dulu belum ada. responsive web design adalah sebuah design website yang bisa diakses oleh seluruh perangkat, mau itu desktop, mobile, maupun tablet. jadi website tidak perlu domain yang berbeda beda, misalnya untuk desktop ada facebook.com, lalu di mobile ada m.facebook.com. dengan responsive web design, pengguna cuma perlu mengakses facebook.com untuk mengakses mau itu di desktop, mobile, dan tablet. lalu ada media query, yaitu salah satu fitur css untuk menerapkan styling atau gaya untuk menerapkan design web terhadap layar tertentu. biasanya sekarang developer mengerjakan bagian dengan layout mobile terlebih dahulu, baru dilanjutkan dengan desktop. hal ini dikarenakan sekarang itu pengguna lebih banyak mengakses melalui perangkat mobile
